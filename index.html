Frequently Asked Questions

Everything you need to know about NeuPro.

What is neumorphism? +

Neumorphism (also called soft UI) is a design trend that combines flat design with skeuomorphism. It uses subtle shadows — both inset and outset — to create a soft, extruded plastic look where UI elements appear to push out from or sink into the background surface.

Is NeuPro free to use? +

Yes, NeuPro is completely free. No signup, no installation, no credit card. Just open it in your browser, design your components, and export the code. Use it for personal or commercial projects.

What export formats are supported? +

NeuPro supports three export formats: HTML with clean inline CSS, React JSX components with useState hooks, and Tailwind CSS class-based markup. All exports are production-ready.

Can I customize the shadows and colors? +

Absolutely. Use the sliders to adjust border radius, shadow distance, and blur radius in real-time. You can also pick custom primary and accent colors with the built-in color pickers.

Do the components work in dark mode? +

Yes. NeuPro includes 6 curated themes — including Dark and Neon Night — that automatically adjust shadow colors for proper contrast. The exported code includes CSS variables for easy theme switching.
